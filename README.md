# WearVault Public Docs

Public support and privacy pages for WearVault.

## Files

- `index.md` - landing page with links
- `support.md` - app support page
- `privacy-policy.md` - privacy policy for App Store Connect
- `_config.yml` - basic GitHub Pages settings

## Publish On GitHub Pages

1. Create a GitHub repository named `wearvault-public-docs`.
2. Push this folder to the repository.
3. In GitHub, open `Settings` -> `Pages`.
4. Set the source to `Deploy from a branch`.
5. Choose `main` and `/ (root)`.
6. Save and wait for the Pages URL.

Then use:

- `.../support.html` for `Support URL`
- `.../privacy-policy.html` for `Privacy Policy URL`
